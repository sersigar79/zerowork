# 📸 Mis Álbumes — Galería Inteligente de Fotos y Vídeos

**Mis Álbumes** es una aplicación web diseñada para organizar, visualizar y disfrutar tus fotos y vídeos de forma elegante, moderna y totalmente automática.  
Puedes subir archivos desde tu ordenador, disco duro externo o cualquier carpeta, crear álbumes personalizados y generar galerías con diseño profesional, efectos visuales y música de fondo.

---

## ✨ Características principales

### 🖼️ Subida de fotos y vídeos
- Arrastra tus archivos a la web o selecciónalos desde tu ordenador.
- Compatible con imágenes y vídeos de cualquier tamaño.
- Puedes subir cientos de archivos a la vez.

### 📚 Creación de álbumes
- Crea álbumes con nombre personalizado (ej. *Viaje a Japón 2025*, *Verano en Cádiz*).
- Añade una descripción para cada álbum.
- La primera foto subida se convierte automáticamente en la **portada del álbum**.
- Organización automática por fechas.

### 🎞️ Galería con diseño atractivo
- Vista tipo mosaico con miniaturas elegantes.
- Efecto de zoom al pasar el ratón.
- Información de cada archivo: nombre, fecha y tipo.
- Modo **blanco y negro** con un clic.

### 🎬 Pase de diapositivas (Slideshow)
- Reproduce tus fotos y vídeos en secuencia automática.
- Resalta la imagen activa con un borde dorado.
- Transiciones suaves cada 3 segundos.
- Ideal para presentaciones o recuerdos familiares.

### 🎵 Música de fondo
- Activa o pausa la música mientras ves tus álbumes.
- La música se reproduce en bucle durante el pase de diapositivas.

### 📊 Gráfica automática
- Muestra cuántas fotos y vídeos se tomaron cada día.
- Perfecto para ver la intensidad de un viaje o evento.
- Gráfica generada con **Chart.js**.

### 📂 Lista de álbumes tipo “tarjetas”
- Vista estilo Netflix con tarjetas de cada álbum.
- Muestra número de archivos y rango de fechas.
- Permite cambiar de álbum con un clic.

### ⬇️ Descarga del álbum (concepto)
- Botón para descargar el álbum completo.
- Preparado para integrar librerías como **JSZip** si quieres generar ZIP reales.

## 🚀 Cómo usar la aplicación

### 1. Subir fotos y vídeos
Puedes hacerlo de dos formas:
- **Arrastrar y soltar** tus archivos en la zona marcada.
- Pulsar el botón **“Subir fotos y vídeos”** y seleccionar archivos desde tu ordenador.

La web acepta:
- Fotos (JPG, PNG, HEIC…)
- Vídeos (MP4, MOV…)

### 2. Crear un álbum
Pulsa **“Crear álbum”** y escribe:
- Nombre del álbum (ej. *Viaje a Bali*)
- Descripción (ej. *Vacaciones en familia, agosto 2024*)

El álbum aparecerá en la lista y podrás seleccionarlo.

### 3. Añadir archivos al álbum
Cada archivo que subas se asignará automáticamente al álbum actual.  
La primera foto se usará como **portada**.

### 4. Ver la galería
Al seleccionar un álbum:
- Verás todas las fotos y vídeos ordenados por fecha.
- Podrás activar el modo blanco y negro.
- Podrás reproducir un pase de diapositivas.
- Podrás activar música de fondo.

### 5. Ver estadísticas del álbum
Debajo de la galería verás una gráfica con:
- Número de fotos por día.
- Fechas de mayor actividad.

### 6. Descargar el álbum
Pulsa **“Descargar álbum”** para iniciar la descarga.  
(La versión actual muestra un aviso; puedes integrar JSZip para generar ZIP reales.)
